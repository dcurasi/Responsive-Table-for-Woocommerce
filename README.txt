Make responsive the Cart and Wishlist table of Woocommerce.


== Description ==

Table for Responsive Woocommerce is a plugin that allows you to make responsive, so easy and fast, the tables of the cart and the wishlist of woocommerce. It is based on the [WordPress Plugin Boilerplate] (https://github.com/DevinVinson/WordPress-Plugin-Boilerplate) and uses [No more tables] to make responsive tables.

It is compatible with the wishlist of [Yith WooCommerce Wishlist].


= Available Languages =

* English - UNITED KINGDOM


== Installation ==

1. Unzip the downloaded zip file.
2. Upload `dc-rtfw` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the \'Plugins\' menu in WordPress
4. Go to Woocommerce > Responsive Table for change the default options
